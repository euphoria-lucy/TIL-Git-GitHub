# 🍀 GitHub 추가 팁

<br>

## 🧸 OctoTree

- GitHub 레포지토리의 디렉토리를 보다 편하게 브라우징
- 크롬 익스텐션 (엣지에서 사용 가능)

<br>

## 🧸 GitHub CLI

- GitHub 작업 전용 CLI 툴

<br>

### ⛄ 로그인 / 로그아웃

```bash
gh auth (login/logout)
```

<br>

### ⛄ 레포지토리들 보기

```bash
gh repo list
```

<br>

### ⛄ 프로젝트 클론

```bash
gh repo clone (사용자명)/(레포지토리명)
```

<br>

### ⛄ 프로젝트 생성 / 삭제

```bash
gh repo (create/delete)
```

<br>

### ⛄ 이슈 목록 보기

```bash
gh issue list
```

<br>

### ⛄ 이슈 열람 / 닫기

```bash
gh issue (view/close) (이슈 번호)
```

<br>

### ⛄ 이슈 생성

```bash
gh issue create
```

<br>

### ⛄ 풀 리퀘스트 만들기 / 목록 보기

```bash
gh pr (create/list)
```

<br>

### ⛄ 풀 리퀘스트 보기 / 코멘트 / 닫기 / 병합

```bash
gh pr (view/comment/close/merge) (PR 번호)
```
