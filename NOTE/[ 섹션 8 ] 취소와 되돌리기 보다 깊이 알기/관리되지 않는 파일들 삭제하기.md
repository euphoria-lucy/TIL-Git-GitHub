# 🍀 관리되지 않는 파일들 삭제하기

<br>

## 🧸 git **clean**

Git에서 추적하지 않는 파일들 삭제

| 옵션 | 설명                                    |
| ---- | --------------------------------------- |
| -n   | 삭제될 파일들 보여주기                  |
| -i   | 인터렉티브 모드 시작                    |
| -d   | 폴더 포함                               |
| -f   | 강제로 바로 지워버리기                  |
| -x   | ⚠️ `.gitignore` 에 등록된 파일들도 삭제 |
