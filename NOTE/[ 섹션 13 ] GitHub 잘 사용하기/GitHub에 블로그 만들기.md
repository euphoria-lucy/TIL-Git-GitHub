# 🍀 GitHub에 블로그 만들기

## 🧸 **GitHub Pages** 사용하기

- 계정별 무료 웹 호스팅

<br>

## 🧸 나의 GitHub Page 만들기

<br>

### ⛄ 레포지토리 생성

- 레포지토리명 : `(내 아이디).github.io` 로 지을 것!
- 로컬로 클론
  <br>

## 🧸 최상위 디렉토리에 `index.html` 작성

- 💡 VS Code 팁 : `!` 입력하고 엔터 누르면 기본 HTML 템플릿 생성
- 내용 작성 뒤 push

<br>
